## 📝 Описание

Современный адаптивный лендинг для умной бутылки SmartHydro, который поможет:
- Повысить конверсию продаж
- Представить продукт профессионально
- Обеспечить отличный UX на всех устройствах

## 🚀 Особенности

- ✅ **Полностью адаптивный дизайн** - работает на телефонах, планшетах и десктопах
- ✅ **Современный UI/UX** - градиенты, тени, плавные анимации
- ✅ **Оптимизированная производительность** - быстрая загрузка
- ✅ **Кроссбраузерная совместимость** - работает во всех современных браузерах
- ✅ **Семантическая разметка** - доступность и SEO-friendly

## 📱 Разделы лендинга

1. **Шапка с навигацией** - быстрый доступ ко всем разделам
2. **Герой-секция** - мощный призыв к действию
3. **Преимущества** - ключевые особенности продукта
4. **Как работает** - простое объяснение функционала
5. **Отзывы** - социальное доказательство
6. **Цены и акция** - убедительное предложение
7. **FAQ** - ответы на частые вопросы
8. **Футер** - контакты и соцсети

## 🛠️ Технологии

- **HTML5** - современная семантическая разметка
- **CSS3** - без использования фреймворков
- **Vanilla JavaScript** - легкие интерактивные элементы
- **Flexbox/Grid** - современная верстка
- **CSS Animations** - плавные переходы и эффекты
